# HighLights-automation
## Features
- Magazine not supported.
- Expect bugs.
- Auto refresh when the unactive.
- auto read books in Books and Phonics section.
- If expect any issue consider visiting Issues tab.
## How to use :
- Install TamperMonkey extension
- Right-click on the extension
- Click 'Manage extension'
- Enable 'Allow User Scripts ' & 'Allow access to file URLs'
- Click into the extension and go to dashboard
- Paste the HighLights-automation.js content into the editor.
